# Real Estate Growth CRM - Marketplace Vertical Solution Kit

This folder turns the real estate offer into a one-time Zoho Vertical Studio build that can be packaged as a Zoho Marketplace vertical solution.

## Build Goal

Create a CRM-only vertical application for real estate brokerages:

> Real Estate Growth CRM helps brokerages capture every property inquiry, assign it to the right agent, track buyer and seller opportunities, manage property showings, and give brokers clear visibility into follow-up, pipeline, and agent performance.

## Recommended V1 Scope

- Zoho Vertical Studio / CRM for Verticals application
- CRM-only package
- Target market: small and mid-size brokerages with 3-25 agents
- No external integrations in V1
- No Zoho Creator portal in V1
- No Zoho Books finance automation in V1
- Advanced integrations sold as custom implementation after install

## Files

1. `01-build-spec.md` - exact Vertical Studio setup instructions.
2. `02-module-field-blueprint.md` - modules, fields, relationships, picklists, and layouts.
3. `03-workflows-dashboards.md` - automations, tasks, alerts, reports, and dashboards.
4. `04-marketplace-listing.md` - Marketplace positioning, description, benefits, screenshots, and install notes.
5. `05-qa-publish-checklist.md` - testing and review checklist before Marketplace submission.
6. `demo-data/` - import-ready sample records for screenshots and QA.
